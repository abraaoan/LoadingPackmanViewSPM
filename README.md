# LoadingPacmanViewSPM

An example of SWIFTUI shape animation loading.

Example application:
https://github.com/abraaoan/LoadingPacmanView


# Usage
```swift
import LoadingPacmanView

var body: some View {
    LoadingView()
    Text("Pac-man loading...")
        .foregroundColor(Color(red: 25 / 255,
                               green: 25 / 255,
                               blue: 166/255))
}
```
